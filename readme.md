# Setting up chat GPT-4 api for idea generation
## What is generative AI?
Generative artificial intelligence is artificial intelligence capable of generating text, images, or other media, using generative models. Generative AI models learn the patterns and structure of their input training data and then generate new data that has similar characteristics. Generative AI can be used for voices, images, videos, or text. With the explosion of the AI industry many APIâ€™s and products exist to assist in generating content for platforms and companies. ## Installing powershell scripts
1. Download both requirements.ps1 and pythonInstall.ps1
2. Run .\pythonInstall.ps1
3. Closes powershell and relaunch to create python instance for packages
4. Run .\requirements.ps1
